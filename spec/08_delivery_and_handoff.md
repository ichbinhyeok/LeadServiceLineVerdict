# 08 Delivery And Handoff

## Workstreams
- Utility and program source collection
- File-based data pipeline and entity normalization
- Page generation and templating
- CTA routing and monetization
- Analytics and index controls

## Milestone 1
- Utility registry
- Source-evidence ledger
- First route inventory
- First utility pages
- Test and filter CTAs

## Milestone 2
- Replacement-cost pages
- Program pages
- Buyer and seller decision pages
- Freshness audit support

## Milestone 3
- Sponsor inventory
- Utility lookup flows
- Conversion optimization

## Acceptance criteria
- A new agent can build from this folder without chat history
- The site can build entirely from versioned `CSV` and `JSON` files without a runtime database
- No local page is indexed without an official source
- Utility pages outperform generic city pages in early cohorts
- CTA routing changes by line-status state

## Handoff order
1. Data and sourcing owner
2. Backend and templating owner
3. SEO and indexing owner
4. Monetization owner
